# Específicaciones del tractor

Es un tractor 6403


- Motor PowerTech™ turbo cargado de 109 hp
- Transmisión SyncShuttle™ 12F/4R
- Disponible con rodados altos y angostos para cultivos de alto valor
- Compatible con sistemas de Autoguiado y conectividad JDLink™
- Potencia nominal de la TDF 96 hp (71.4 kW)
- Moderno diseño con mejor visibilidad y de fácil mantenimiento. 

- Tiene un consumo de entre 4.5 litros y 8.7 litros de disel por hora
- Con capacidad de 152 L
